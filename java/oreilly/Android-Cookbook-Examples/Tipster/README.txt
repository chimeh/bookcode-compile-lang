Full app: compute and show restaurant bill with tip

This version was re-created from the book files rather
than from the original author's download, because his
web site went away.

We've added + and - buttons.

There are some known issues: 

The "Calculate" button doesn't get 
enabled at the correct time sometimes.

On some devices the virtual keyboard hides
the computed results, so you have to hit
the Back button.
