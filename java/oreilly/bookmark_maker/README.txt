----前書き----
本ツールはオライリー・ジャパン様で購入できる電子書籍のPDFフ
ァイルからしおりを挿入した新たなPDFファイルを生成するツールで
す。

しおりはPDFファイル内の目次から自動生成されます。

新たに生成されるファイルはPDFの保護機能が解除されていますの
で扱いには気をつけてください。

本ツールはαバージョンのため、バグが多く含まれている可能性が
あります。

今のところ作者が試している状態では動物が表紙の書籍で動くこと
を確認しています。Head Firstシリーズでは上手く動きません。


----使い方----
1.bookmark_maker.jarをダブルクリックし、しおりを挿入したいPDF
ファイルを選択してください。

2.新しく生成されるファイル名を指定してください。
デフォルトでは"既存PDFファイル名-new.pdf"が指定されます。

3.開始ページ数を入力してください。
開始ページは書籍内で記述されているページ数ではなく、PDFファ
イルのページ数になります。
開始ページ数は書籍内で目次の記述が終わるページ数から-1をし
た数字です。

4.目次挿入開始ボタンをクリックしてください。
